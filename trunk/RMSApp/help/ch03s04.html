File Monitor

Monitors files.

Local/Remote Modes

The agent supports both local and remote modes.

Configuration

The following configuration options are available:

  • Root folder: The folder which is the root of the monitored file structure.
  • File name pattern: The regular expression that is used to filter monitored files.
  • Ignore subfolders: If true only files will be processed; if false the processing will be recursive down the folder tree.

Contents

For each monitored file, the agent provides the size and the date when the file was last modified.
