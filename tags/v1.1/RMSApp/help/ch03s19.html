Sun Application Server

This agent returns performance metrics for a Sun Application Server.

Local/Remote Modes

The agent supports both local and remote modes.

Supported Versions

The agent supports Sun Application Server versions 8.x

Configuration

The following configuration options are available:

  • Username: user name to use when connecting to Sun Application Server
  • Password: password to use when connecting to Sun Application Server
  • Connection string: the JMX URL used to connect to Sun Application Server
  • Show only runtime data: if true only runtime mbeans are displayed

Figure 23. Configuration sample (Sun Application Server 8.x)

Contents

All data available from Sun Application Server via JMX is accessible; the runtime data is not structured but a dashboard is provided that contains some of the most useful data views.
