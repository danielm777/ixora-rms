Weblogic

This agent returns performance metrics for a Weblogic Application Server.

Local/Remote Modes

The agent supports both local and remote modes.

Supported Versions

The agent supports Weblogic Application Server versions 8.x and 9.x

Configuration

The following configuration options are available for Weblogic version 8.x:

  • Username: user name to use when connecting to Weblogic Application Server
  • Password: password to use when connecting to Weblogic Application Server
  • Weblogic home folder: the folder where Weblogic is installed (eg. C:\bea\weblogic8)
  • Classpath: path to jar files required by this agent; all are relative to the Weblogic home folder above
  • Connection string: the connection string required to establish the JMX connection
  • Show only runtime data: if true only runtime mbeans are displayed
